# ICLR 2026

| Paper Title | Brief Description |
| --- | --- |
| Reinforced Latent Reasoning for LLM-based Recommendation | Introduces LatentR3, which uses reinforcement learning to learn compact latent reasoning and improve LLM-based recommendation without explicit chain-of-thought supervision. |
| More Than What Was Chosen: LLM-based Explainable Recommendation Beyond Noisy User Preferences | Proposes C-APO to jointly model Revealed Preference and Coherent Preference, improving recommendation quality and generating more persuasive rationales. |
| Token-Efficient Long-Term Interest Sketching and Internalized Reasoning for LLM-based Recommendation | Proposes SIREN, which compresses long-term user interests and internalizes reasoning to greatly reduce inference latency while maintaining strong performance. |
| Discrete Diffusion for Bundle Construction | Presents a discrete diffusion framework for non-sequential bundle construction and completion, improving recommendation quality for unordered bundle scenarios. |
| Generalizing Linear Autoencoder Recommenders with Decoupled Expected Quadratic Loss | Introduces DEQL to extend the closed-form solution space of linear autoencoder recommenders and achieve better recommendation performance under a broader hyperparameter range. |
| Gradient-Based Diversity Optimization with Differentiable Top-$k$ Objective | Introduces a differentiable top-k diversity objective that can be integrated into recommendation training to improve diversity with minimal relevance loss. |
